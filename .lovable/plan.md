

# Пайстартер — Страница объекта недвижимости

## Обзор
Вёрстка детальной страницы объекта недвижимости для маркетплейса паёв «Пайстартер» по макету из Figma. Только фронтенд, без бэкенда. Данные захардкожены, фото — placeholder'ы.

---

## Дизайн-система

- **Цвета:** Светлый фон (#FFFFFF), тёмная секция (тёмно-синий/чёрный ~#1A1B2E), акцентный голубой для кнопок и активных элементов, серый для вторичного текста
- **Типографика:** Крупные жирные заголовки (serif или sans-serif как в макете), body-текст ~16px
- **Скругления:** Крупные (16-20px) для карточек и изображений
- **Иконки:** lucide-react (подберу максимально похожие на макет)

---

## Секции страницы (сверху вниз)

### 1. Хедер
- Логотип «Пайстартер» (текстовый placeholder)
- Поисковая строка «Поиск по объектам»
- Кнопка «Личный кабинет» с иконкой пользователя

### 2. Заголовок объекта
- Хлебные крошки: ЗПИФН «Пайстартер двадцать один» · Рейтинг: A · Поделиться
- Крупный заголовок: «Терминал Чкаловский, Екатеринбург»
- Ключевые показатели в ряд: Стоимость объекта, Стоимость за м², Площадь, Доходность (с иконкой замка)

### 3. Галерея фотографий
- Сетка изображений: большое фото слева + 4 маленьких справа
- Кнопка «Показать все» на последнем фото
- Placeholder-изображения

### 4. Характеристики + Боковая панель
- Левая часть: описание объекта, информация об арендаторе, преимущества (иконки + текст)
- Правая «липкая» панель: цена за 1 пай (120 364₽), кнопка «Презентация», кнопка «Купить пай»

### 5. Расположение
- Карта (placeholder-изображение или встроенная карта)
- Адрес + список преимуществ локации
- Два информационных блока: «Будущий транспортно-торговый хаб» и «Стабильный спрос на складские площади» с фото

### 6. Финансовые показатели (тёмная секция)
- Тёмный фон, белый текст
- Табы слева: «Доход на пай» (активный), «Стоимость пая», «Оборот паев»
- График (area chart) с переключателем периода «12 месяцев / Всё время»
- Итого доход, тултипы на графике
- Две кнопки: «Купить паи», «Связаться с менеджером»

### 7. Похожие объекты
- 3 карточки в ряд
- Каждая: фото с бейджем рейтинга (A+, B+), название, город, площадь, срок, стоимость, кол-во паев, доходность, ISIN

### 8. Вопросы и ответы (FAQ)
- Нумерованный аккордеон (01-04)
- Вопросы раскрываются по клику

### 9. Контактная форма
- Заголовок «Остались вопросы? Напишите нам»
- Поля: Имя, Email, Телефон, Сообщение
- Чекбокс согласия на обработку данных
- Кнопка «Отправить сообщение»

### 10. Футер (тёмный)
- Логотип + юридическая информация (ОГРН, ИНН)
- Телеграм-канал
- Колонки ссылок: Категории, О компании, Контакты
- Копирайт + ссылки на политики

---

## Технические детали
- Компонентный подход: Header, PropertyHero, PhotoGallery, PropertyDetails, LocationSection, FinancialSection, SimilarProperties, FAQ, ContactForm, Footer
- Все данные захардкожены (моковые данные)
- Графики через recharts (уже установлен)
- FAQ через Accordion (уже установлен)
- Адаптивная вёрстка (desktop-first, потом адаптируем)
- Изображения — placeholder'ы, которые легко заменить
